## todoister add

Añade un nuevo recurso a Todoist (actualmente: project, task).


### Flags globales:

<dl>
  <dt><code>-t</code>, <code>--token</code> <code>&lt;string&gt;</code></dt>
  <dd>usa <code>&lt;string&gt;</code> como token de la API de Todoist</dd>
</dl>

### Comandos

* [todoister add project](todoister-add-project.md)	 - Añade un nuevo proyecto
* [todoister add task](todoister-add-task.md)	 - Añade una nueva tarea a un proyecto
